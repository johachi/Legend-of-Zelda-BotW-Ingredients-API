Legend of Zelda - Breath of the Wild API

Listing Ingredients

You can list all ingredients in the database by sending a get request to /api/ingredients. You can also choose to get information from only one ingredient by visiting /api/ingredients/:nameOrId. :nameOrId should be replaced with the index number of the item in the database, or by the ingredients exact name.

Adding Ingredients

You can add an ingredient to the database by sending a post request to /api/ingredients. You can also choose to get information from only one ingredient by visiting /api/ingredients/. The body of the post request should send the data as a JSON file. It should contain the following keys: item_name as string, type as string, hearts as string, effect as number, potency as string, duration as number,, resale_value as number.

Editing Ingredients

You can edit an ingredient in the database by sending a patch request to /api/ingredients/:nameOrId. The body of the patch request should contain the data to be edited as a JSON file. It should contain one or several of the following key-value pairs: item_name as string, type as string, hearts as string, effect as number, potency as string, duration as number,, resale_value as number.

Removing Ingredients

You can edit an ingredient in the database by sending a patch request to /api/ingredients/:Id. The Id should be the Id in the database.
